# Contributing to Bobaland

Thank you for your interest in contributing to Bobaland!

## Getting Started

1.  **Fork** the repository on GitHub.
2.  **Clone** your fork locally.
3.  Create a **new branch** for your feature or fix.

## Guidelines

-   **Code Style**: Please follow the existing style in shell scripts. Use 4 spaces for indentation.
-   **Documentation**: Update relevant documentation if you change functionality or keybindings.
-   **Commits**: Use descriptive commit messages.

## Pull Requests

1.  Push your branch to GitHub.
2.  Open a Pull Request against the `main` branch.
3.  Describe your changes clearly.

## Reporting Issues

If you find a bug, please open an issue in the issue tracker with details on how to reproduce it.
