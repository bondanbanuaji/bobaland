## Window Manager: Hyprland
- **Config**: `~/.config/hypr/hyprland.conf`
- **Why**: Best performance, smooth animations, highly customizable.
- **Key Features**:
  - Blur and transparency
  - Tiling layout (dwindle)
  - Gestures support

## Status Bar: Waybar
- **Config**: `~/.config/waybar/`
- **Style**: Floating pil style with modules for:
  - Workspaces
  - Clock/Date
  - Audio/Brightness
  - Network/Bluetooth
  - System Tray

## Application Launcher: Wofi
- **Config**: `~/.config/wofi/`
- **Style**: Minimalist, perfectly centered.

## Terminal: Kitty
- **Config**: `~/.config/kitty/kitty.conf`
- **Features**:
  - GPU acceleration
  - Ligatures support
  - Image preview support

## Notifications: Dunst
- **Config**: `~/.config/dunst/dunstrc`
- **Style**: Matches the Miku theme (colors, border radius).

## Lock Screen: Swaylock
- **Config**: `~/.config/swaylock/config`
- **Features**:
  - Blurred background screenshot
  - Ring indicator
