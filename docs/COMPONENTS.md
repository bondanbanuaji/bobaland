# Components Overview

## Core System
- **Hyprland**: A dynamic tiling Wayland compositor based on wlroots that doesn't sacrifice on its looks.
- **Waybar**: Highly customizable Wayland bar for Hyprland, Sway and other wlroots based compositors.
- **Kitty**: A modern, feature-rich, GPU based terminal emulator.

## Utilities
- **Wofi**: A launcher/menu program for wlroots based wayland compositors.
- **Dunst**: Lightweight notification daemon.
- **Wlogout**: Wayland based logout menu.

## Theming
- **SDDM**: Simple Desktop Display Manager.
- **Plymouth**: Graphical boot animation.
- **GRUB**: Bootloader theming.
